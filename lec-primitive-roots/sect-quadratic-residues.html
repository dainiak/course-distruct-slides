A quadratic residue in field \(\bbF\) is an element \(y \in \Fstar\), such that \(y=x^2\) for some \(x\in\bbF\).
The Legendre symbol \(\chi\) is a function \(\bbF\to \{-1,\,0,\,1\}\), where \[\chi(x)=\begin{cases} \phantom{-}0,\,\text{if } x = 0, \\ -1,\, \text{if } x \text{ is a quadratic non-residue},\\ \phantom{-}1,\, \text{if } x \text{ is a quadratic residue}. \end{cases}\]
Number of quadratic residues
In any field \(\bbF\) where \(2\notdivides\card\bbF\), exactly half of the elements in \(\Fstar\) are quadratic residues.
Multiplicativity of the Legendre symbol
When \(2\notdivides\card\bbF\), for any \(a,b\in\bbF\) we have \(\chi(ab)=\chi(a)\cdot \chi(b)\).

This follows from the fact that if \(\alpha\in\bbF\) is a primitive element, then
\(\alpha^0,\,\alpha^2,\,\dots\) are quadratic residues,
\(\alpha^1,\,\alpha^3,\,\dots\) are quadratic non-residues.
