Lemma "\(>0\implies =\euler\)"
If there are elements of order \(k\) in \(\Fstar\), then there are exactly \(\euler(k)\) of them.
Proof.

Let \(\ord a = k\). Let \(A:=\{a^0,\,\dots,\,a^{k-1}\}\).

Let \(P(x):= x^k - 1\). For \(a^j\in A\) we have \[ P(a^j) = (a^j)^k - 1 \next{ = (a^k)^j - 1 } \next{ = 1^j - 1 = 0, } \]

that is \(A\subseteq \roots(P)\).

Since \(|A|=k=\deg P\), then \(\roots(P) = A\).

Let \(\ord a = k\). Let \(K:=\{a\in\Fstar:\: \ord a = k\}\). Then \(K\subseteq \{a^0,\,\dots,\,a^{k-1}\}\).

If \(m\le k\) and \(\GCD(m,\,k) = 1\), then \[\forall\ell\; (\ell \lt k \implies (a^m)^\ell \neq 1),\] otherwise we would have \(a^{\ell m} = 1\) while \(\ord a \notdivides \ell m\).

It is now clear that \[K=\{a^m\mid \GCD(m,\,k) = 1\},\] and therefore, \(\card K = \euler(k).\)

Sum of values of Euler's function
For any \(n\) we have \[ \sum_{k:\:k\divides n}\euler(k)=n. \]
Proof: reduce fractions!

We have \(n=\card\calN\), where \[\calN=\bigl\{\tfrac{1}{n},\,\tfrac{2}{n},\,\dots,\,\tfrac{n}{n}\bigr\}\]

We see that exactly \(\euler(k)\) elements of set \(\calN\) can be represented as irreducible fractions with denominator \(k\) (and, of course, \(k\divides n\)).

Let \(\calO:=\{\ord a\mid a\in \Fstar\}\), and \(n:=\card\Fstar\).

Obviously, \(\calO\subseteq D_n = \{ d:\: d\divides n \}\). \[ |\Fstar| = \sum_{k\in \calO} \euler(k) \next{\green{\le} \sum_{k\in D_n} \euler(k)=\card\Fstar,} \] and therefore, \(\calO=D_n\).

Theorem
In any field \(\bbF\) there exists a primitive element, that is, such an \(\alpha\in\Fstar\) that \[ \Fstar = \{\alpha^0,\,\alpha^1,\,\dots,\,\alpha^{\card\Fstar-1}\}. \]
Proof.

The number of elements of order \(\card\Fstar\) equals \(\euler(\card\Fstar)\ge 1\).

If we fix a primitive element \(\alpha\) in a finite field, then for any \(x\in \bbF\setminus\{0\}\) we can properly define the discrete logarithm: \(\log_\alpha x \), which has some properties of the usual logarithm:
  • \(\log_\alpha (xy)=(\log_\alpha x + \log_\alpha y)\vstep{?}\next{\bmod \card\Fstar,}\)
  • \(x^{\log_\alpha y}=y^{\log_\alpha x},\)
  • \(\log_\alpha x = (\log_\beta x \cdot \log_\alpha \beta) \bmod\card\Fstar.\)
