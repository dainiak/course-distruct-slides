Количество элементов заданного порядка
Если в \(\Fstar\) есть элементы порядка \(k,\) то их ровно \(\phi(k).\)
Доказательство.

Пусть \(\ord a = k\). Пусть \(A:=\{a^0,\,\dots,\,a^{k-1}\}\).

Для \(a^j\in A\) имеем \[ \step{ (a^j)^k - 1 } \step{ = (a^k)^j - 1 } \step{ = 1^j - 1 } \step{ = 0, } \]

то есть каждый \(a^j\in A\) — корень многочлена \(x^k-1\). Так как \(|A|=k=\deg (x^k-1)\), то т. Лагранжа имеем, что \(A\) — это множество всех корней этого многочлена.

Сумма значений функции Эйлера
Для любого \(n\) имеем \[ \sum_{k:\:k\divides n}\euler(k)=n \]
Доказательство: сокращаем дроби!
\[n=\{\tfrac{1}{n},\,\tfrac{2}{n},\,\dots,\,\tfrac{n}{n}\}=\calN\]

Видим, что ровно \(\euler(k)\) элементов множества \(\calN\) представимы несократимыми дробями со знаменателем \(k\) (и, конечно, \(k\divides n\)).

Пусть  \(K:=\{\ord a\mid a\in \Fstar\}\),  и  \(n:=|\Fstar|\).

Очевидно, \(K\subseteq D_n = \{ d:\: d\divides n \}\). \[ |\Fstar| = \sum_{k\in K} \euler(k) \pstep{\green{\le} \sum_{k\in D_n} \euler(k)=|\Fstar|,} \] а стало быть, \(K=D_n\).
